# 📚 BookSky

BookSky is a simple frontend web application that allows users to write and manage reviews about the books they have read.
The goal of this project is to create a clean and interactive interface where users can keep track of their reading experiences.

## 🚀 Features

* Add a book review
* Display books and reviews in a clean layout
* Simple and user-friendly interface
* Dynamic UI interactions using JavaScript

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📂 Project Purpose

This project was built as a frontend practice project to improve skills in building responsive web interfaces and handling dynamic user interactions.

## ▶️ How to Run

1. Clone the repository
2. Open the project folder
3. Open `index.html` in your browser

## 📌 Future Improvements

* Add edit and delete functionality for reviews
* Store reviews using local storage or a database
* Improve UI design and responsiveness
